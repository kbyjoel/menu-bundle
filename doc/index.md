# Aropixel Menu Bundle Documentation

Welcome to the Aropixel Menu Bundle documentation. This bundle allows you to manage menus (headers, footers, etc.) for your front office directly from the back office.

## Table of Contents

* [Installation](installation.md)
* [Entity Customization](entities.md)
* [Custom Data Sources](sources.md)

## Overview

The Aropixel Menu Bundle provides a flexible way to build nested menus using a drag-and-drop interface. It supports various item types out of the box (links, pages, sections) and can be easily extended with custom data sources.
